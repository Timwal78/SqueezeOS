# 07 — Nexus402

**Status:** ✅ Live · **Source:** `/nexus402`

## Overview
Nexus402 is the agent-facing marketplace and RAG (retrieval-augmented generation) layer — a Next.js application that lets AI agents and developers discover, query, and pay for ScriptMasterLabs products and signal data through a unified storefront and knowledge interface.

## Architecture
- **Frontend:** Next.js marketplace UI — product listings, agent onboarding, x402 wallet flow.
- **RAG layer:** indexes product docs (including this architecture map), Pine Script indicator catalog, and signal definitions so agents can query "what does X product do / cost / return" in natural language before paying.
- **Bridge** (`core/nexus402_bridge.py`): connects Nexus402's marketplace listings to 402Proof for payment settlement and to the core engines for live signal previews.

## Agent & Human Access
- **MCP server**, JSON-RPC 2.0 (listed in `agents.json`).
- **Marketplace:** browsable product catalog with x402 checkout per listing.
- **RAG queries:** natural-language product discovery — "find an indicator for squeeze detection," "what does 402Proof cost per call."
