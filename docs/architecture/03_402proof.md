# 03 — 402Proof

**Status:** ✅ Live · **Source:** `/402proof` (Go) · **Live:** https://four02proof.onrender.com

## Overview
402Proof is the shared payment-rail and compliance gateway for the entire ecosystem. Every x402-gated product (SqueezeOS, NEXUS-402, Ghost Layer Marketplace, Agent Credit Marketplace) settles through it. It combines RLUSD/XRPL settlement with a built-in FICO-style "agent credit bureau" and a usage-based discount curve.

## Architecture
- **Gateway** (`cmd/`, `internal/`, `middleware/`): verifies XRPL payments, issues signed access tokens (HMAC-SHA256, sub-ms local verification).
- **proof402-middleware** (`/middleware/js`, npm `proof402-middleware`) and **Python** middleware (`/middleware/python`) — drop-in x402 middleware for Express/Next.js/Cloudflare Workers and Python services respectively.
- **Bureau** (`core/bureau_client.py`): tracks agent payment history → produces a credit score used for discount-curve pricing and credit-marketplace eligibility.
- **Discount curve**: per-second, usage-weighted pricing — heavier/repeat agent users earn lower marginal rates, administered via the bureau.

## Agent & Human Access
- **MCP:** `https://four02proof.onrender.com/mcp` — 11 tools.
- **Flow:** `GET /api/{endpoint}` → `402 + payment terms` → pay RLUSD (XRPL) or USDC (Base) → retry with `X-PAYMENT` / `X-Payment-Token` → `200`.
- **npm:** `npm install proof402-middleware` (v1.2.0) — Express / Next.js / Cloudflare Workers.

## x402 Flow (3 steps)

```
1. POST /v1/invoice   { endpoint_id }
   ← { invoice_id, pay_to, amount, memo_hex, expires_at, payment_options[] }

2. Send payment on XRPL/Xahau/Base with memo_hex embedded

3. POST /v1/verify    { invoice_id, tx_hash, agent_wallet }
   ← { access_token (JWT), receipt_id, loyalty_tier, free_credits }
   
   access_token → use in X-Payment-Token header on protected endpoint
```

## Agent Credit Bureau (FICO-style)

Score range: 300 (no history) → 850 (institutional tier).  
Factors: payment history, spend volume, call frequency, KYB tier, risk events, loyalty tier.  
Grades: A (750+), B (650–749), C (550–649), D (below 550).  
Attestation JWT: 24h TTL, portable, verifiable by third parties at `/v1/bureau/verify-attest`.

## Loyalty Tiers (automatic)

Bronze → Silver → Gold → Platinum → Diamond, based on cumulative RLUSD spend.  
Each tier unlocks free credits and reduced firewall friction.  
KYB (Know Your Bot) elevation available via admin: `basic` (-10 risk), `verified` (-20 risk).
