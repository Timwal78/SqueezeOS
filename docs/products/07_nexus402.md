# 07 — Nexus402 (Next-Gen Agent Marketplace + RAG Council)

**Live URL:** https://nexus402.onrender.com

**Language:** Next.js (TypeScript) + Python backend
**Deploy:** Render (cloud, 24/7)

---

## What It Does
Next-generation agent marketplace and RAG (Retrieval-Augmented Generation) council layer. Built on top of Ghost Layer's credit marketplace infrastructure. Adds a rich Next.js frontend, MCP server, and an OpenMythos RAG knowledge layer for agent-to-agent commerce and knowledge exchange.

## Key Components
- `nexus402/app/` — Next.js frontend
- `nexus402/backend_server.py` — Python API backend
- `nexus402/mcp-server/` — MCP tools for agent integration
- `nexus402/openmythos-rag/` — RAG knowledge layer
- `nexus402/ghost-layer/` — Ghost Layer integration
- `nexus402/ghost-layer-rpc/` — RPC client for Ghost Layer
