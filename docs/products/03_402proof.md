## What It Does
The payment backbone of the entire stack. Two products in one service:

**Rail A — XRPL/RLUSD Invoice Flow**
Issues signed XRPL invoices, verifies on-chain payment, returns JWT for gated endpoints. The proprietary rail for all RLUSD transactions across the stack.

**Rail B — FICO-Style Agent Credit Bureau**
300–850 credit score for every agent wallet. Score drives rent discounts (Dream Pool), fee discounts (Futures), marketplace access tier, and loyalty tier benefits. Failure-closed: offline bureau → 0% discount (never blocks a call).

## Endpoints

```
POST /v1/invoice                         → issue XRPL RLUSD invoice
POST /v1/verify                          → verify on-chain payment → JWT
GET  /v1/bureau/score?wallet=rXXX        → 300–850 credit score
GET  /v1/bureau/report?wallet=rXXX       → full credit report
GET  /v1/bureau/score/{wallet}           → score for bureau_client.py TTL-cached lookup
```

## Bureau Score Factors
- Payment history (on-time, expired, cancelled)
- Loyalty tier (DIAMOND = −30 risk premium)
- Total lifetime volume

## Bureau Score → Discount Curve
| Score | Discount |
|-------|----------|
| 300–499 | 0% |
| 500–599 | 5% |
| 600–699 | 10% |
| 700–799 | 15% |
| 800–850 | 20% (max) |

Platform always retains ≥ 80% of base fee.

## Links To
