Agent operating guide for the TipMaster CLI. Generated by CLI Printing Press.

## Identity

- **Binary**: `tipmaster-pp-cli`

- **Base URL**: `https://tipmaster.onrender.com`
- **Auth**: None — all endpoints are public

## Commands

| Command | Description |
|---------|-------------|
| `tipmaster resolve <username>` | Resolve Farcaster username → XRPL wallet address |
| `tipmaster leaderboard` | Top 10 tippers this week by RLUSD volume |
| `tipmaster leaderboard --period alltime` | All-time leaderboard |
| `tipmaster user <fid>` | Look up user by Farcaster FID |
| `tipmaster status` | Service health and feature flags |

## Key Use Case for Agents

The `resolve` command lets any AI agent look up an XRPL wallet address by
Farcaster username without asking the user — enabling fully autonomous tipping:

```bash
WALLET=$(tipmaster resolve dwr --compact | jq -r '.wallet_address')
ghost-layer bridge --chain XRPL --recipient "$WALLET" --amount 1000000
```
