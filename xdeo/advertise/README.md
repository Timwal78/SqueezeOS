# xDEO — Advertising Kit

Everything you need to promote xDEO. Nothing here is required for the app to run — it's pure marketing material.

## What's in this folder

| File | What it is | How to use it |
|------|-----------|---------------|
| `LAUNCH_KIT.md` | Copy-paste ad text — tweets, launch thread, Reddit/Discord posts, one-liners | Open it, copy a block, paste it where you're posting |
| `INFO_SHEET.md` | The "what is xDEO" fact sheet — the elevator pitch + every key number | Send to anyone who asks "what is this?" |
| `HOW_TO_LAUNCH.md` | Plain-English, click-by-click guide to the 2 GitHub buttons you need to press | Follow it top to bottom once |
| `flyer_print_8.5x11.png` | Full-page printable flyer with QR codes baked in | Print it, or post the image online |
| `qr_share_page.png` | QR → the live demo + docs page | Put on slides, flyers, anywhere |
| `qr_mcp_endpoint.png` | QR → the MCP endpoint (for AI-agent people) | For technical audiences |
| `qr_github.png` | QR → the GitHub repo | For developers |
| `x_banner_1500x500.png` | A spare X/Twitter banner (you already have one you like — this is a backup) | Optional |
