⚡ MCP Quickstart

Add to Claude Desktop claude_desktop_config.json — works with any MCP client.

{
  "mcpServers": {
    "squeezeos": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-fetch",
               "https://squeezeos-api.onrender.com/mcp"],
      "env": {
        "Authorization": "Bearer sml_live_YOUR_KEY_HERE"
      }
    }
  }
}

Or call REST directly: curl -X POST https://squeezeos-api.onrender.com/api/council -H "Authorization: Bearer sml_live_..." -d '{"symbol":"NVDA"}'
